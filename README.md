# Stock Portfolio Tracker

## Project Overview
This project is a Python-based Stock Portfolio Tracker developed as part of my internship task.

The program allows users to enter stock names and quantities, calculates total investment, and provides a summary of the portfolio.

## Features
- User input for stock name and quantity
- Uses dictionary to store predefined stock prices
- Calculates total investment value
- Displays a clean portfolio summary
- Option to save results into a CSV file

## Technologies Used
- Python
- Dictionary
- Functions
- File Handling (CSV)

## How to Run
1. Install Python
2. Run the program:

python stock_tracker.py

## Output
The program displays the portfolio summary and saves the result into a CSV file.
